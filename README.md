🍳 CookIt — Умный кулинарный помощник
CookIt — это мобильное Android-приложение, которое помогает превратить продукты из вашего холодильника в полноценные блюда. Больше не нужно ломать голову над вопросом "Что приготовить?", нейросеть сделает это за вас.

✨ Основные функции
Генерация рецептов по ингредиентам: Просто введите список того, что у вас есть, и получите подборку блюд.

Умная сортировка: Сначала приложение показывает рецепты, для которых у вас есть все ингредиенты, а затем — те, где нужно что-то докупить.

Интеграция с ИИ: Использует современные модели (Llama 3.1 / Qwen) через Hugging Face API для создания реалистичных и вкусных рецептов.

Детализация: Каждый рецепт включает время приготовления, калорийность и пошаговую инструкцию.

🛠 Технологии
Язык: Kotlin

IDE: Android Studio

Сеть: OkHttp / Retrofit (для связи с Hugging Face API)

Формат данных: JSON (динамический парсинг рецептов)

🚀 Как запустить
Клонируйте репозиторий:

Bash
git clone https://github.com/ваш_логин/cookit.git
Создайте файл local.properties в корне проекта (если его нет) и добавьте ваш токен Hugging Face:

Properties
HF_TOKEN=hf_ваш_токен
Откройте проект в Android Studio и запустите его на эмуляторе или реальном устройстве.

📝 Пример работы API (JSON формат)
Приложение общается с моделью и получает структурированные данные:

JSON
{
  "name": "Бургер с говядиной",
  "calories": "550 ккал",
  "cook_time": "20 мин",
  "steps": ["Поджарьте котлету...", "Соберите бургер..."]
}
